# ✨ Features

* 🌍 **Real-Time Weather**

  * Current weather condition
  * Live temperature

* 🔍 **Search Any Location**

  * Search weather by city name.
  * Quickly retrieve weather information for locations around the world.

* 🎨 **Dynamic Weather Backgrounds**

  * Automatically changes the app background according to the current weather.
  * ☀️ Sunny
  * ☁️ Cloudy
  * 🌧️ Rainy
  * ⛈️ Stormy
  * ❄️ Snowy
  * 🌫️ Foggy

* 📱 **Material Design UI**

  * Clean and modern Android interface.
  * Smooth user experience.

* ⚡ **Fast & Lightweight**

  * Quick API responses.
  * Optimized performance.

# 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/SkyView.git
```

2. Open the project in **Android Studio**.

3. Let Gradle sync all dependencies.

4. Connect an Android device or start an emulator.

5. Click **Run ▶** to build and launch the application.

---

# 🌦️ How It Works

1. Launch the SkyView app.
2. Search for a city using the search bar.
3. The app requests live weather data from the **Go Weather API**.
4. Weather details are displayed instantly.
5. The application's background changes dynamically to match the current weather conditions, creating a more immersive user experience.
